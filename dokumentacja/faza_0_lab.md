# Faza 0: Inkubator SOK w Laboratorium Cyfryzacji

Faza 0 projektu SOK (System Ochrony Konstytucji) nie jest etapem biernego oczekiwania, lecz okresem intensywnej budowy fundamentów technologicznych i organizacyjnych w modelu inkubatora.

## Cele Fazy 0

1. **Budowa MVP Silnika**: Stworzenie działającego prototypu silnika rezolucji w Pythonie.
2. **Walidacja na PoC**: Udowodnienie skuteczności logiki SOK na istniejących, prostych ustawach (waloryzacja, odsetki).
3. **Zbudowanie Społeczności**: Pozyskanie pierwszych 10-20 partnerów komercyjnych i akademickich.
4. **Zabezpieczenie Finansowania**: Uruchomienie zbiórki startowej i mechanizmu 5% od wdrożeń pobocznych.

## SOK jako Projekt Flagowy Laboratorium

W Fazie 0 SOK korzysta z "osłony" Laboratorium Cyfryzacji. Oznacza to, że:
- Programiści pracujący nad systemami ERP dla biznesu współtworzą biblioteki logiczne wykorzystywane w SOK.
- Naukowcy zajmujący się NLP (Natural Language Processing) szkolą modele na tekstach prawnych Konstytucji.
- Infrastruktura serwerowa jest współdzielona, co minimalizuje koszty startowe.

## Harmonogram Inkubacji (12 miesięcy)

| Miesiąc | Działanie | Cel |
| :--- | :--- | :--- |
| **M1-M2** | Rejestracja Fundacji, start repozytorium | Formalizacja i widoczność open-source. |
| **M3-M4** | Budowa modułu pól semantycznych | Umożliwienie translacji pojęć prawnych na predykaty. |
| **M5-M6** | Realizacja PoC (Proof of Concept) | Raport publiczny: SOK weryfikuje ustawę o waloryzacji. |
| **M7-M9** | Formalizacja projektu Konstytucji | Pierwsza pełna weryfikacja logiczna nowej Konstytucji. |
| **M10-M12** | Kampania informacyjna i zbiórka | Przygotowanie gruntu pod Fazę 1 (legislacyjną). |

## Rola Wolontariatu i Mikro-zadań

Inkubacja opiera się na rozproszonym zaangażowaniu:
- **Deweloperzy**: Zadania programistyczne (Python/FastAPI).
- **Prawnicy/Logicy**: Definiowanie reguł i weryfikacja predykatów.
- **Tłumacze**: Praca w systemie Transmem (nagradzana e-Talarami).

## Wyjście z Inkubatora

Faza 0 kończy się w momencie, gdy system posiada:
1. Działający silnik zdolny do weryfikacji niesprzeczności.
2. Sformalizowaną wersję nowej Konstytucji.
3. Stabilny strumień finansowania z modelu partnerskiego lub zbiórki.
